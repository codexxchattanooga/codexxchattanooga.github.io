Smooth Scrolling Using jQuery

Scroll to anchor [1]...

Smooth scrolling can easily be added to anchors with jQuery. The link is given a class “scroll” and the target div is given an id which is referenced in the anchor like this...

<a href="#bottom" class="scroll">Scroll to the bottom of the page...</a>

The target div...

<div id="bottom">Text Goes Here</div>

Include jQuery in the HTML doc head and add this script just before the closing body tag...

<script type="text/javascript">

jQuery(document).ready(function($) {
$(".scroll").click(function(event){
event.preventDefault();
$('html,body').animate({scrollTop:$(this.hash).offset().top}, 2000);
});
});
</script>

The animation speed in this instance is “2000”. Reduce it to speed up the animation and increase it to slow things down.
